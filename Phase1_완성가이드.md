폴더로 이동\ncd naver-crawler\ndir  # (Windows)\n# 또는\nls   # (Mac/Linux)\n```\n\n**확인할 파일:**\n- ✓ scraper.py\n- ✓ config.json\n- ✓ requirements.txt\n- ✓ README.md\n\n### 2단계: 의존성 설치\n\n**Windows:**\n```bash\npip install -r requirements.txt\n```\n\n**Mac/Linux:**\n```bash\npip3 install -r requirements.txt\n```\n\n**설치 확인:**\n```bash\npip list | grep -E \"requests|beautifulsoup4|lxml\"\n```\n\n### 3단계: 키워드 설정\n\n**config.json 파일을 텍스트 에디터로 열기:**\n\n```json\n{\n  \"keywords\": [\n    \"당신의_검색어_1\",\n    \"당신의_검색어_2\",\n    \"당신의_검색어_3\"\n  ],\n \"Python 3.12\"\n  ],\n  ...\n}\n```\n\n### 4단계: 실행\n\n**Windows:**\n```bash\npython scraper.py\n```\n\n**Mac/Linux:**\n```bash\npython3 scraper.py\n```\n\n### 5단계: 결과 확인\n\n**생성된 파일:**\n```\n✓ results.json   - 구조화된 데이터\n✓ results.csv    - 엑셀 호환\n✓ results.md     - 마크다운 형식\n✓ crawler.log    - 실행 로그\n```\n\n**결과 보기:**\n```bash\n# 로그 확인\ncat crawler.log\n\n# JSON 결과 보기 (Windows)\ntype \"키워드3\"\n  ],\n  \n  // ⚙️ 검색 동작 설정\n  \"search_settings\": {\n    // 키워드당 크롤링할 최대 결과 수\n    // 범위: 1-50 (낮을수록 빠르고 안전)\n    \"max_results\": 10,\n    \n    // 키워드 요청 사이의 대기 시간 (초)\n    // 범위: 1-10 (크면 안전, 작으면 빠름)\n    \"request_delay\": 2,\n    \n    // 네이버 응답 대기 시간 (초)\n    // 범위: 5-30 (느린 네트워크면 크게)\n    \"timeout\": 10\n  },\n  \n  // 🌐 User-Agent (봇 차단 우회)\n  \"user_agent\": \"Mozilla/5.0 \"제목\",\n      \"URL\",\n      \"요약\",\n      \"검색일\",\n      \"키워드\"\n    ]\n  },\n  \n  // 📝 로깅 설정\n  \"logging\": {\n    // 로그 레벨: DEBUG, INFO, WARNING, ERROR\n    \"level\": \"INFO\",\n    // 로그  \"request_delay\": 3,\n    \"timeout\": 15\n  }\n}\n```\n\n**특징:**\n- 매우 안전\n- 느린 속도 (1-2분)\n- 차단될 확률 낮음\n\n#### 상황 2: 일상적인 사용 (균형)\n\n```json\n{\n  \"keywords\": [\"AI\", \"웹개발\", \"Python\"],\n  \"search_settings\": {\n    \"max_results\": 10,\n    \"request_delay\": 2,\n    \"timeout\": 10\n  }\n}\n```\n\n**특징:**\n- 균형잡힌 설정\n- 보통 속도 (2-3분)\n- 권장 사항\n\n#### 상황 3: 빠른 처리 (고급)\n\n```json\n{\n  \"keywords\": [\"AI\", \"웹\", \"Python\", \"JS\", \"React\"],\n  \"search_settings\": {\n    \"max_results\": 15,\n    \"request_delay\": 1,\n    \"timeout\": 8\n  }\n}\n```\n\n**특징:**\n- 빠른 속도
